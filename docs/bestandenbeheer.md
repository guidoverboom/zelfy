# Bestandenbeheer

Dit document biedt een overzicht van de mappenstructuur en een referentie over welk script/bestand waarvoor dient binnen het project.
*(Houd deze altijd up to date indien er nieuwe mappen of bestanden bij komen)*

## Mappen
- `/archief/` : Opslag voor behandelde procedure bestanden zoals `implementation_plan.md` en `walkthrough.md`.
- `/concepten/` : Locatie voor experimentele componenten, proefcode en prototypes. 
- `/docs/` : Centrale map voor alle project metadata, logging en logica.

## Bestanden
- `agents.md` : Bevat de kerninstructies voor de AI agent gedragingen. Lees hiermee altijd de huidige fase af.
- `docs/changelog.md` : Applicatie updates en code wijzigingen.
- `docs/action_log.md` : Overzicht van alle menselijke en AI gestuurde acties.
- `docs/actielijst.md` : Todo-lijst, bevat openstaande acties waar de agent bij start aan moet herinneren.
- `docs/description.md` : Algemene pitch/doel van de te ontwikkelen app.
